# Hi there! I'm Natalie 👋

QA Engineer specialising in test automation | Playwright | TypeScript | Manchester, UK

## About Me

I'm a QA Engineer with hands-on experience in test automation and functional 
testing across web, mobile (React Native), and API-based applications in a 
fintech environment. I transitioned into tech after 5 years as a physiotherapist 
in Hong Kong — completing the Northcoders full-stack bootcamp before moving into 
QA. I bring systematic analytical thinking and attention to detail to everything 
I test.

Currently building automation frameworks with Playwright and TypeScript, exploring 
AI-assisted testing with Playwright MCP, and always looking to embed quality 
earlier in the development lifecycle.
